# Excel Data Cleaning and Formatting Project

## 📌 Project Overview
This project demonstrates essential data cleaning and formatting techniques in Microsoft Excel. The objective was to improve data quality, ensure consistency, and prepare the dataset for analysis.

---

## 🛠 Tasks Completed

### 1. Handling Missing Values
- Identified blank or missing entries in the dataset.
- Replaced missing values where appropriate.
- Ensured the dataset remained complete and usable for analysis.

### 2. Removing Duplicates
- Detected duplicate records within the dataset.
- Removed duplicate entries using Excel's **Remove Duplicates** feature.
- Improved data accuracy and reliability.

### 3. Splitting and Merging Data
#### Splitting Data
- Separated combined information into multiple columns using **Text to Columns**.

#### Merging Data
- Combined related columns into a single field using concatenation functions.

### 4. Number Formatting
- Applied appropriate formats to numerical data.
- Used formats such as:
  - Currency
  - Percentage
  - Date
  - Decimal Numbers
- Improved data readability and presentation.

### 5. Conditional Formatting
- Applied formatting rules to highlight important data.
- Used conditional formatting to:
  - Identify missing values
  - Highlight duplicate entries
  - Emphasize key values and trends

---

## 📊 Results
After completing the data cleaning process:

- ✅ Missing values were handled appropriately
- ✅ Duplicate records were removed
- ✅ Data fields were split and merged where required
- ✅ Numerical values were properly formatted
- ✅ Important data patterns were highlighted using conditional formatting

---

## 🧰 Tools Used
- Microsoft Excel
- Text to Columns
- Remove Duplicates
- CONCAT / Concatenation Functions
- Number Formatting
- Conditional Formatting

---

## 🎯 Conclusion
This project demonstrates how Excel can be used to clean, organize, and format data effectively. These techniques help improve data quality and prepare datasets for accurate analysis and reporting.
